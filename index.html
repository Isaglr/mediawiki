MediaWiki

O que é?

MidiaWiki é um sistema computacional livre, com o uso da linguagem PHP, e funciona como um gestor de conteúdo web usando a base de dados MySQL no projetos principais da fundação, como a Wikipédia.

Para que serve?

É uma plataforma de software usada para criar e gerenciar wikis, incluindo o famoso site Wikipedia. Ela é útil para possibilitar a criação de wikis colaborativos onde os usuários podem editar e contribuir com o conteúdo.

Como funciona?

Uma plataforma de software que permite a criação e gerenciamento de wikis colaborativos, ela organiza as contribuições dos usuários, mantém um histórico de edições e oferece recursos de pesquisa e navegação para facilitar o acesso ao conteúdo.

O que é possível criar com ele?

Permite aos usuários editar páginas, adicionar conteúdo e colaborar em tempo real, mantendo um histórico de edições e oferecendo recursos de pesquisa e navegação.
